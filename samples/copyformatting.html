Formatting document using Copy Formatting Documentation

The optional Copy Formatting feature is provided through the Copy Formatting plugin which by default is available in the Full distribution.

When the plugin is enabled, the   button is automatically added to the toolbar. You can use it to copy formatting from one place in the document and apply it to the other.

To copy styles, place your cursor inside or select styled part of the text and press the   button. Then click inside some word to style that word or select part of the text to apply formatting to it. You can also use Ctrl+Shift+V keystroke to apply formatting.

Note that not all styles are copyable and removable, e.g. Copy Formatting would not copy styles from links and won't remove links when you apply formatting to them. There is also Copy Formatting sticky mode that allows you to copy formatting once and apply it to more than one place. To enable that mode, double click the   button or use Ctrl+Shift+C keystroke. The sticky mode could be switched off by clicking the   button or by pressing Escape key.

Lists and tables

Copy Formatting also allows to copy formatting between tables and lists. In case of lists, it's even able to change the list's type, e.g. applying formatting from bulleted list to the numbered one should change the type of numbered one into the buletted list.

Filter customization

As it was stated before, Copy Formatting does not copy formatting from certain type of elements. By default these elements are:

  • links
  • images
  • iframes
  • widgets
  • media embeds
  • headings

However that setting could be altered by two configuration rules: CKEDITOR.config.copyFormatting.allowRules and CKEDITOR.config.copyFormatting.disallowRules. You can for example allow to copy formatting only from the bold text (placed inside <strong> tags).

Cursor customization

In the previous examples you probably noticed that the Copy Formatting changes the cursor inside the editor. Additionaly the cursor for the whole webpage outside of the editor is also changed to the "disabled" cursor that indicates that you can't apply formatting there. This cursor could be switched off using the CKEDITOR.config.copyFormatting.outerCursor configuration variable.

More advanced modifications of cursors used by CopyFormatting could be done by adding custom styles to the editor's stylesheet.

Keystrokes customization

You can also change the default keystrokes for copying (Ctrl+Shift+C) and applying formatting (Ctrl+Shift+V). They can be changed by updating values of CKEDITOR.config.copyFormatting_keystrokeCopy and CKEDITOR.config.copyFormatting_keystrokePaste configuration variables.
